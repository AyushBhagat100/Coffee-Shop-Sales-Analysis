# Coffee-Shop-Sales-Analysis

Project Overview:-

This project focuses on analyzing coffee shop sales data using Microsoft Excel and Power BI to generate meaningful insights and interactive dashboards. The goal is to understand sales performance, customer behavior, and business trends to support data-driven decision-making.

Objectives:-
- Analyze overall sales performance
- Identify top-selling products
- Track daily, monthly, and yearly trends
- Understand customer purchasing patterns
- Create an interactive dashboard for business insights

Tools & Technologies:-
- Microsoft Excel 
- Power BI 

Dataset Description:-
The dataset contains information related to:
- Transaction Date
- Product Name
- Category
- Quantity Sold
- Unit Price
- Total Sales

Data Cleaning (Excel):-
- Removed null/missing values 
- Fixed date formats
- Standardized column names
- Created calculated fields (e.g., Total Sales = Quantity × Price)

Dashboard Features (Power BI):-
- Sales trend over time (daily/monthly)
- Top-selling products
- Category-wise sales distribution
- Total revenue and key KPIs
- Filters for dynamic analysis

Key Insights:-
- Identified peak sales periods
- Found best-performing products
- Analyzed customer buying trends
- Highlighted revenue contribution by category
